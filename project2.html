Full-stack Data Management Application using Python Django Framework

  • Live Demo
  • Source code

This Data Management Application does not cover customer relations or enterprise resource planning processes, rather designed to solve and improve specific operations related processes by creating an application to store and manage data. The application creation process uses a web-application development methodology like “agile” and uses currently broadly used programming languages like Python and JavaScript.

The process of creating the application completed in following steps:

  • Designing and testing data structure in PostgresSQL
  • Implementing the data structure in Django models
  • Creating server side of the application with Django views
  • Designing client side with HTML, CSS and JavaScript
  • Dashboard design with Chart.js

Application is deployed on Azure cloud platform and uses Azure Database for PostgreSQL.

Application is designed to manage data related to the following processes:

  • Valve Testing Operations
  • Valve Inventory Management
  • Order Management
  • Shipping Management
